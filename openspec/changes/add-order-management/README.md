# README: 订单管理核心 (Order Management Core)

## 任务摘要
相位 2 实现了系统的核心业务逻辑：订单管理。通过建立多维度的订单配置和严格的 SOP 审核流程，系统能够支持从合同签订到物流发货的全流程数字化交付。

## 关键成就
- **高度结构化的配置**: 支持为每台订单下的设备定制精细的硬件（电源、线缆、主机）和软件（商户信息、登录账户）参数。
- **按台设备与 SN 追踪**: 订单设备按台拆分并生成 SN，生产中心按设备进度推进。
- **SOP 流转体系**: 将生产过程拆解为 6 个标准阶段（装配 -> 部署 -> 调试 -> 打包 -> 物流 -> 完成），并通过审核机制确保每个环节的合规性。
- **财务记录完善**: 支持付款记录创建、状态与备注编辑。
- **订单概述一致性**: 订单概述与数量由设备清单自动汇总，列表与详情保持一致。
